# Karim Wael | 3D Portfolio

A high-performance interactive 3D portfolio showcasing the projects, skills, and experience of Karim Wael, Software Engineer & AI Systems specialist.

## ✨ Features

- **Interactive 3D Keyboard** — Spline keyboard representing technical skills like C++, React, Node.js, and Python.
- **Dynamic Animations** — Powered by GSAP and Framer Motion for smooth scroll and hover reveals.
- **Space Theme Aesthetic** — Dark mode cosmic vibe optimized for performance.
- **Clean Architecture** — Next.js 14, React 18, and Tailwind CSS.
- **Contact Integration** — Direct messaging via Resend.

## 🛠️ Tech Stack

| Layer | Technologies |
|---|---|
| **Framework** | Next.js 14, React 18, TypeScript |
| **Styling** | Tailwind CSS, Shadcn UI, Aceternity UI |
| **Animation** | GSAP, Framer Motion |
| **3D** | Spline Runtime |
| **Email** | Resend |

## 🚀 Getting Started

1. Clone the repository and install dependencies using `pnpm install` or `npm install`.
2. Set up environment variables locally by filling out `.env.local` with your Resend API key.
3. Run `npm run dev` to launch the development server on `http://localhost:3000`.
